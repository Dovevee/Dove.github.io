## Github Learning
1. 基本概念：
    - Repository(仓库):仓库用来存放项目代码，每个项目对应一个仓库
    - Star(收藏)
    - Fork(复制克隆项目)：(fork项目独立存在)
    - Pull request(PR)
    - Watch:(可收到项目的更新提醒）
    - Issue

2. 仓库管理：
    - 新建文件
    - 编辑文件
    - 删除文件
    - 上传文件
    - 搜索文件:(快捷键T)
    
3. Github issue：
    - New issues
